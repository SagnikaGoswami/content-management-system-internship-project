# Content Management System (Internship Project)
This Project was developed during my Full Stack Development Internship at Ardent Computech Pvt Ltd as part of a team of four members.

## Project Description
The Content Management System (CMS) is a web application that enables administrators and users to manage website content through a dashboard. The system includes role-based access where admins can create and manage user accounts, while users can create, edit, and delete posts, update their profiles, and interact through comments. Profile deletion requires admin approval to ensure proper control and security.
